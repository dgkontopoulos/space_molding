NAME

space_molding

DESCRIPTION

A novel space aided homology modeling approach.

SYNOPSIS (running command)

space_molding.pl pdb_file

DEPENDENCIES

-Perl, >= 5.10

-Bio::PDB::Structure, >= 0.01 (Perl module)

-File::Spec, >= 3.33 (Perl module)

-List::Util, >= 1.25 (Perl module)

-Math::Round, >= 0.06 (Perl module)

EXAMPLES

   -Visualisation (with superposition and slab)

   

   -Output PDB file example (logo)

   1YI5_model.pdb - model of the a-cobratoxin-AChBP complex


SUPPORTED OPERATING SYSTEMS

space_molding runs out of the box -via install.sh- on GNU/Linux and OSX systems (reasonably up to date ones, that is).

It should run out of the box -via install.sh- on BSD systems, as well.

It can run on Windows systems, as long as you manually install Perl and the other dependencies listed above. The install.sh script won't work on default Windows systems.
